Get Started in 30 Seconds

Zero configuration required. Just clone, install, and run!

# Clone the repository
git clone https://github.com/roe04u/skinassistant.git
cd skinassistant

# Install dependencies
cd skin_ai_assistant
pip install -r requirements.txt

# Run all services (automatic port allocation)
cd ..
python run_all.py

# Access URLs will be displayed in console
# Typical: http://localhost:8001, 8002, 8003
